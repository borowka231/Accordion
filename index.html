Frequently Asked Questions


Here are some of the frequently asked questions


What is roadmap.sh?

V

roadmap.sh is a community effort to create learning paths,guides, projects ideas and other similar content to help developers grow in their careers.
What are the plans for roadmap.sh?

V

The main plan for roadmap.sh is to become a central learning hub for developers, which includes plans to add more features like quizzes, project ideas, and public profiles to enhance the existing roadmaps, guides, and videos.
How is roadmap.sh built?

V

The website is built with Astro and Tailwind and is deployed to GitHub pages. The project is open-core and the codebase can be found on GitHub
Can i use roadmap.sh in my team?

V

Yes, you can use roadmap.sh in your team, as it is a free, community-based resource for learning paths, and the creator built it to be helpful to developers.
How can I create custom roadmaps?

V

To create custom roadmaps, you can fork the roadmap.sh repository on GitHub, make your desired changes to the roadmaps, and then deploy your customized version using GitHub Pages or another hosting service of your choice.
Is roadmap.sh really 7th most started project on GitHub?

V

Yes, as of September 2021, roadmap.sh is the 7th most starred project on GitHub among developer learning resources, according to the creator's statement.
